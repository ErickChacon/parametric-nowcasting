--- title: "Mapping Food Insecurity in the Brazilian Amazon using a Spatial Item Factor Analysis Model" author: - name: "Erick A. Chacón-Montalván" link: "https://erickchacon.gitlab.io" ---

This web presents all the analysis performed in the paper denominated “Mapping food insecurity in the Brazilian Amazon using a spatial item factor analysis model”. We propose a spatial item factor analysis model, and use it to map food insecure areas in Ipixuna, a remote city in the Brazilian Amazon.

Explore the content of the web using the available sidebar. The content is divided in 5 main sections:

  • Data extraction
  • Data processing
  • Data exploration
  • Data modelling
  • Summarise results

The scripts are used to render the web: https://erickchacon.gitlab.io/food-insecurity-mapping.
